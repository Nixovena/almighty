Almighty Linux 1.0 Beta

First public beta release of Almighty Linux.

  • Initial public release of the hybrid Debian Testing / Kali Linux distribution
  • 49 custom-built OSINT and security tools
  • 30 integrated Kali Linux repository packages
  • Almighty OSINT Browser with fingerprint protection and integrated utilities
  • Hardened kernel configuration (ASLR, Spectre/Meltdown mitigations, BPF hardening)
  • Runtime security via sysctl hardening profile
  • Kernel module blacklisting for attack surface reduction
  • Permission hardening service with boot-time enforcement
  • APT sandbox for secure package management
  • Tor integration, MAC spoofing, hostname randomization tooling
  • Firejail and Bubblewrap application isolation
  • Forensic trace cleaning capabilities
  • MATE desktop environment with Conky monitors
  • Calamares graphical installer
